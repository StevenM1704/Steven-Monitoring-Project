# Contributing

Thank you for your interest in contributing to Steven-Monitoring-Project.

## How to contribute

1. Fork this repository on GitHub.
2. Create a new branch for your change.
3. Make your changes with clear commits and useful messages.
4. Ensure the application runs locally with Docker Compose and on Minikube.
5. Open a Pull Request with a clear description and testing steps.

## Guidelines

- Keep Dockerfiles and YAML files without comments.
- Keep the project structure and naming consistent.
